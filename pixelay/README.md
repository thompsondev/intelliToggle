# Pixelay Comparison Files

This folder should contain:

1. **pixelay-desktop.png** - Desktop overlay screenshot from Pixelay comparison
2. **pixelay-mobile.png** - Mobile overlay screenshot from Pixelay comparison
3. **notes.md** - Notes on differences found (already created)

## How to Generate Pixelay Screenshots

1. Deploy your site or run `npm run dev`
2. Open Pixelay browser extension or tool
3. Load your Figma design
4. Capture desktop overlay (1920x1080 or similar)
5. Capture mobile overlay (375x667 or similar)
6. Save screenshots to this folder
